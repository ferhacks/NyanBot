## Clone this project

```bash
> git clone https://github.com/Samu330/NyanBot
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd termux-wabot
> bash install.sh
```

### Usage
```bash
> npm start
```

### Settings
```json
{
	"prefix": ".",
	"ownerNumber": "529984xxxxx",
	"apiKey": "Your ApiKey here"
}
```
